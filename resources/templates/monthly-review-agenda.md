# 월간 FinOps 리뷰 아젠다

## 1. 회의 개요

### 목적

멀티클라우드(AWS/Azure/GCP) 비용 현황을 정기적으로 점검하고, 최적화 기회를 식별하며, 비즈니스 단위경제 KPI와 연결하여 데이터 기반 의사결정을 지원합니다.

### 빈도 및 일정

| 항목 | 내용 |
|------|------|
| 빈도 | 월 1회 |
| 시점 | 매월 셋째 주 수요일 오전 10:00 |
| 시간 | 60분 |
| 장소 | 본사 대회의실 또는 화상 (Teams) |

### 참석자 역할

| 역할 | 담당자 | 책임 |
|------|--------|------|
| 퍼실리테이터 (진행) | 이지수 (FinOps팀 팀장) | 회의 진행, 의사결정 촉진 |
| 데이터 준비 | 박민준 (FinOps 엔지니어) | 대시보드 업데이트, 리포트 준비 |
| 서비스개발 대표 | 서비스개발팀 리드 | CC-200 비용 현황 설명, 최적화 협업 |
| 데이터/AI 대표 | 데이터/AI팀 리드 | CC-300 비용 현황 설명, ML 최적화 협업 |
| 인프라 대표 | 인프라운영팀 리드 | CC-100 비용 현황 설명, RI 관리 |
| 필요 시 참석 | 재무팀 담당자 | 예산 대비 실적 확인, 이월 예산 조정 |

## 2. 사전 준비 체크리스트

회의 3일 전까지 박민준 엔지니어가 아래 항목을 완료합니다.

### 데이터 준비

- [ ] FOCUS 정규화 데이터 최신화 (`focus-normalized.csv` 당월 데이터 반영)
- [ ] 클라우드 비용 대시보드 업데이트 (AWS Cost Explorer, Azure Cost Management, GCP Billing)
- [ ] CostCenter별 비용 집계 완료 (CC-100 / CC-200 / CC-300)
- [ ] 전월 대비 증감률 계산 완료
- [ ] 이상 비용 탐지 결과 정리 (임계치: 전주 동 기간 대비 +20% 이상)

### 최적화 분석

- [ ] 유휴 리소스 목록 작성 (CPU 사용률 < 5%, 7일 이상 지속)
- [ ] Right-sizing 후보 목록 작성 (CPU/메모리 모두 < 40%, 30일 기준)
- [ ] RI/SP 활용률 계산 (Used / (Used + Unused) × 100)
- [ ] 신규 RI 구매 기회 식별 (가동률 ≥ 80% 이상, RI 미보유 리소스)

### 보고 자료

- [ ] 전월 액션 아이템 완료 현황 업데이트
- [ ] KPI 지표 현황 업데이트 (아래 지표 체크리스트 참조)
- [ ] 회의 자료 사전 공유 (회의 24시간 전까지 Slack #finops-review 채널)

## 6. HBT 맞춤 KPI 예시

### 6.1 MAU당 클라우드 비용

```
MAU당 클라우드 비용 = 월 총 클라우드 지출(KRW) ÷ 월간 활성 사용자 수(MAU)

예시:
- 월 클라우드 지출: ₩320,000,000
- MAU: 2,500,000명
- MAU당 비용: ₩320,000,000 ÷ 2,500,000 = ₩128/MAU

목표: 전월 대비 감소 (절감 최적화 성과 측정)
데이터 소스: AWS Cost Explorer + 내부 MAU 데이터베이스
담당: 박민준 (FinOps) + 서비스개발팀
```

### 6.2 API 호출 100만 건당 비용

```
API 호출 100만 건당 비용 = API 관련 월 클라우드 비용(KRW) ÷ (API 호출 건수 ÷ 1,000,000)

예시:
- API 관련 월 비용: ₩45,000,000 (CC-200 중 API 프로젝트)
- 월 API 호출: 3,000,000,000건 (30억 건)
- 100만 건당 비용: ₩45,000,000 ÷ 3,000 = ₩15,000/100만 건

목표: 전월 대비 감소 (서비스 효율성 측정)
데이터 소스: API Gateway CloudWatch Metrics + AWS Cost Explorer
담당: 박민준 (FinOps) + 서비스개발팀
```

### 6.3 ML 모델 학습 1회당 비용

```
ML 모델 학습 1회당 비용 = ML 관련 월 GCP 비용(KRW) ÷ 해당 월 학습 실행 횟수

예시:
- ML 관련 월 GCP 비용: ₩25,000,000
- 월 학습 실행: 50회
- 학습 1회당 비용: ₩25,000,000 ÷ 50 = ₩500,000/회

목표: 전월 대비 감소 (ML 효율성 측정)
데이터 소스: GCP Billing + Vertex AI 학습 로그
담당: 박민준 (FinOps) + 데이터/AI팀
```
